# 汽车信息平台 - 全平台爬虫实现说明

## 更新内容

### 1. 爬虫模块 (scraper.js)
- ✅ 支持汽车之家、懂车帝、易车三个平台
- ✅ 反爬机制：User-Agent轮换、随机延迟、请求头伪装
- ✅ 图片校验：MD5、尺寸检测
- ✅ 增量更新与全量同步
- ✅ 自动去重与数据更新

### 2. 定时任务 (scheduler.js)
- ✅ 汽车之家新闻：每2小时
- ✅ 懂车帝新闻：每3小时
- ✅ 易车新闻：每4小时
- ✅ 车型数据：每天凌晨3点
- ✅ 完整同步：每天中午12点
- ✅ 数据校验：每天凌晨4点

### 3. 测试模块 (test-suite.js)
- ✅ 爬虫模块测试
- ✅ API接口测试（健康检查、品牌、新闻、车型、统计）
- ✅ 前端渲染测试（Playwright E2E测试）
- ✅ 图片校验测试
- ✅ 数据质量测试

## 使用方法

### 1. 安装依赖
```bash
cd backend
npm install
```

### 2. 启动服务
```bash
npm run dev
```

### 3. 手动触发同步
```bash
# 全部平台
npm run scrape

# 仅新闻
curl -X POST http://localhost:3001/api/sync/news

# 仅车型
curl -X POST http://localhost:3001/api/sync/models
```

### 4. 运行测试
```bash
npm test
```

## 注意事项

1. **反爬限制**：真实网站可能有更严格的反爬机制，如遇403可考虑：
   - 使用代理IP池
   - 增加延迟时间
   - 使用无头浏览器（Playwright/Puppeteer）

2. **图片版权**：抓取的图片仅供展示学习使用，商业使用需注意版权

3. **频率控制**：遵守目标网站的robots.txt，合理设置请求频率
